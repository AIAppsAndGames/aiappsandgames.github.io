File Transfer - Sharing

Fast P2P File Sharing. Zero Internet Required.

Get it on Google Play

Description

Transmit large videos, music folders, applications, and documents directly between devices. Powered by Wi-Fi Direct for high-speed offline connections.

Key Features

  • • 50MB/s Speed: Send movies and folders up to 200 times faster than standard Bluetooth.
    • \n
  • • Cross-Platform Browser Sharing: Generate local web portals to connect with Windows, Mac, or Linux.
    • \n
  • • AES-256 Handshakes: Secure local validation keys prevent network snooping.

Android Permissions & Data Safety

Required Permissions

  • • NEARBY_WIFI_DEVICES / ACCESS_FINE_LOCATION: Required to discover nearby peer devices over local Wi-Fi nodes.
    • \n
  • • READ_EXTERNAL_STORAGE: Select files from your library to transmit.

Data Safety

Files are transmitted directly from one device to another. We do not operate transit servers, and no data is synced online.
